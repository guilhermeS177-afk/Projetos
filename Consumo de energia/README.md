# ⚡ Calculadora de Consumo de Energia

## 📌 Sobre o projeto
Este projeto foi desenvolvido em Python com o objetivo de calcular o consumo mensal de energia elétrica de um aparelho, com base em seu uso diário.

## 🚀 Tecnologias utilizadas
- 🐍 Python
- 💻 Git & GitHub

## 📊 Calculo utilizado.

consumoMensal = (potencia * horasDia * 30) / 1000

## 💰 Cálculo de custo
Foi considerado o valor de R$ 0,75 por kWh.
